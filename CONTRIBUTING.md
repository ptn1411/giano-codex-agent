# Contributing to Giano Codex Agent

## Branching Strategy

- **Format**: `feature/<taskId>-<short-description>`
- **Example**: `feature/gca-stabilize-01-fix-regex`

## Commit Messages

- **Format**: `<taskId>: <summary>`
- **Example**: `gca-stabilize-01: fix regex in safety.ts`

## Pull Requests

- **Title**: `<taskId> <Task Title>`
- **Example**: `gca-stabilize-01 Fix regex syntax in safety.ts`

## Workflow

1. Pick a task from Giano.
2. Create a branch following the naming convention.
3. Implement changes.
4. Verify with `npm run check`.
5. Commit and push.
6. Create PR.
